# OptimalStudyDesign
This is the code for the simulation studies in *Simulation-based Power Analysis for Sequential Multiple Assignment Randomized Trials*, preprint available soon.

The `OptimalDesign_RCode_ToShare.R` file includes the code for the simulations using the real pilot Sequential Multiple Assignment Randomized Trial (SMART) data and the real external trial data. However, these data are not available and the variables should be replaced with variables from the user's use-case. The file is organized as follows: loading necessary libraries, helper functions, and finally the simulations. (Data preparation steps are not included as data are not publicly available.)
